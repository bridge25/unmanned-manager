## 주제 유형별 렌즈 매핑

### Type Detection Rules

```python
# 키워드 기반 자동 감지
def detect_topic_type(topic: str) -> list[str]:
    types = []

    # Business 키워드
    if any(k in topic for k in ["수익", "사업", "마케팅", "판매", "비즈니스", "돈", "매출"]):
        types.append("business")

    # Tech 키워드
    if any(k in topic for k in ["개발", "구현", "자동화", "API", "코드", "기술", "시스템"]):
        types.append("tech")

    # Investment 키워드
    if any(k in topic for k in ["투자", "주식", "코인", "자산", "포트폴리오", "수익률"]):
        types.append("investment")

    # Trend 키워드
    if any(k in topic for k in ["트렌드", "시장", "미래", "전망", "성장", "동향"]):
        types.append("trend")

    return types if types else ["general"]
```

## Business 렌즈

**적용 시점**: 수익 모델, 사업 기회, 마케팅 전략 분석

### 필수 분석 항목

| 렌즈 | 질문 | 데이터 소스 |
|------|------|------------|
| **시장성** | 시장 규모는? 성장률은? | 시장 보고서, 통계 |
| **ROI** | 투자 대비 수익은? | 사례 분석, 벤치마크 |
| **경쟁** | 경쟁자는 누구? 차별화는? | 경쟁사 분석 |
| **진입장벽** | 시작하기 얼마나 어려운가? | 실제 사례 |
| **수익모델** | 어떻게 돈을 버는가? | 비즈니스 모델 분석 |

| 렌즈 | 질문 | 데이터 소스 |
|------|------|------------|
| **기술스택** | 어떤 기술이 필요한가? | 기술 문서, 사례 |

**적용 시점**: 트렌드 분석, 시장 전망, 미래 예측

| 렌즈 | 질문 | 데이터 소스 |
|------|------|------------|
| **성장률** | 얼마나 빠르게 성장하나? | 시장 데이터 |
| **포화도** | 시장이 포화 상태인가? | 시장 분석 |

## 복합 렌즈 적용

### 다중 유형 감지 시

주제가 여러 유형에 해당할 경우:

```markdown
## 렌즈 적용: Business + Tech

### Business 관점
[Business 렌즈 적용]

### Tech 관점
[Tech 렌즈 적용]

### 통합 분석
- **Business-Tech 시너지**: [...]
- **Trade-off**: [...]
```

### 렌즈 우선순위

1. 사용자가 명시한 관점 우선
2. 주제 핵심 키워드 기반
3. 사용자 프로필 기반 (예: 개발자 → Tech 가중치 ↑)
