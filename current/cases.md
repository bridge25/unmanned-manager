## Template

아래 형식으로 케이스를 기록합니다. AI가 자동으로 추가합니다.

```markdown
## CXXX: 케이스 제목

| 항목 | 내용 |
|------|------|
| **날짜** | YYYY-MM-DD |
| **상황** | 어떤 상황에서 의사결정이 필요했는지 |
| **선택지** | 어떤 옵션들이 있었는지 |
| **판단** | 어떤 결정을 내렸는지 |
| **근거** | 왜 그렇게 결정했는지 |
| **결과** | ✅ 성공 / ❌ 실패 / ⏳ 대기 |
| **학습** | 이 경험에서 배운 것 |
| **태그** | #태그1 #태그2 |
```
